Research

My research interests lie primarily in Robotics, Control, Machine Learning and Optimization.

RL-CBF-RRTX:Online Learning-Based Safe Critical Motion Planning in Unknown Environments
Huaiyuan Rao*, Kyriakos G. Vamvoudakis
In Progress, 2024
code

This work proposes an online motion planning framework for dynamic environments for systems with nonlinear dynamics. It leverages RRTX for global path planning and replanning to produce waypoints for agent and uses a critic-only value network to online learning a reasonably larger portion of the actual safe set with user predefined rewards and generate safe controller for navigating in an unknown environment.

Multi-section Continuum Robots: Modeling and Control
Huaiyuan Rao*, Guanglin Ji, Qian Gao, Yin Xiao, Zhenglong Sun
In Progress (Review Paper), 2024

This work discusses the advances in the modeling, control field and open challenges of multisection continuum robots.

Towards Optimal Control of Continuum Manipulator Using Reinforcement Learning
Huaiyuan Rao*, Guanglin Ji, Zhenglong Sun
In Progress, 2023
code

This work propose a unified framework that constructing dynamic model of continuum manipulator systems from collected data using Koopman operator and develops RL-LQR, a LQR controller combined with a RL tunner implementing on continuum manipulator under input constraints and train DDPG agent to get the best weight matrix and implemented controller on real robot system.
